# Testing

## What is covered
- Template definitions and integrity.
- `transformPages` behavior for emojis and dividers.
- Create/undo logic with mocked Figma API.

## Commands
- Unit tests: `npm test`
- Typecheck: `npm run typecheck`
- Preflight: `npm run preflight`

## When to update tests
- Any template change.
- Any change to emoji/divider behavior.
- Any change to create/undo flow.
